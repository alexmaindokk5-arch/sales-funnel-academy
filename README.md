# Sales Funnel Academy

Chatter Training Platform for Freya Studios.

## Setup

1. Clone this repo
2. `npm install`
3. Set environment variables (TURSO_URL, TURSO_TOKEN, ADMIN_PW)
4. `npm start`

## Features

- 7-Day structured learning plan
- 50 quiz questions across 5 phases
- Chat simulator
- Manager dashboard with strike system
- Dark/Light mode
- PWA support
